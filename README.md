# CloudPhone PDF Viewer

A lightweight PDF reader widget for CloudMosa CloudPhone (240x320 feature phones).

## Features
- Open local PDF files
- Zoom in/out with `2` and `8`
- Page up/down with `*` and `#`
- Scroll with navigation arrows
- Soft L/R key support for menu/back

**Add to CloudPhone**
1. Go to [CloudPhone Developer Page](https://www.cloudfone.com/my).  
2. Select **Add Widget**.  
3. Paste this Start URL:  
   `https://tasmon.github.io/PDF-Viewer`  
4. Upload icons if required.  
5. Save and refresh — **PDF Viewer** will appear on your CloudPhone.
